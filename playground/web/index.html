What is Sprout?

Sprout is a general-purpose language built entirely from scratch in a single C file — its own lexer, parser, tree-walking interpreter, and garbage collector, with no Node, no LLVM, nothing. It aims to be the kindest language to learn with while still being able to build real software: types & inheritance, first-class functions & closures, pattern matching, regular expressions, a standard library, persistence, modules, a package manager, and a code formatter — all in ~100 KB.

Good to know

The playground runs sandboxed: file, network, and shell builtins are off, and every Run starts with a clean slate. Errors are friendly and tell you the line. For the full language, read the cheat sheet or grab a native binary from Releases.
